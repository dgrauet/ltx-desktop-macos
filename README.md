# LTX Desktop macOS — Agent Team Setup

## Quick Start

```bash
# 1. Copy all files into your project root
mkdir ltx-desktop-macos && cd ltx-desktop-macos
tar xzf ltx-desktop-agent-team.tar.gz

# 2. Launch Claude Code with Agent Teams
export CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1
claude
```

Then give Claude this first prompt:

```
Read CLAUDE.md and AGENTS.md completely. You are the Lead agent. 
Execute Sprint 1 from AGENTS.md — spawn each sub-agent in order, 
wait for dependencies where noted, and gate on the marathon test.
```

That's it. One terminal, one lead, sub-agents spawned automatically.

## Agents

| Agent ID | Role | Workspace | Spawned via |
|----------|------|-----------|-------------|
| (lead) | 🎯 Orchestrator | root | You launch it |
| backend-engine | ⚙️ MLX pipelines, memory, perf | backend/engine/ | `Task(agent="backend-engine", ...)` |
| backend-api | 🌐 FastAPI, endpoints, queue | backend/ | `Task(agent="backend-api", ...)` |
| frontend | 🖥️ SwiftUI app | app/ | `Task(agent="frontend", ...)` |
| qa | 🧪 Tests, scripts, stability | tests/ + scripts/ | `Task(agent="qa", ...)` |

## Sprint Gates

Sprint 1 → Sprint 2: **Marathon test must pass** (10 consecutive generations, memory stable)
Sprint 4 → Sprint 5: **Full test suite must pass**
